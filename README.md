# Setup
```
$ sh setup.sh
$ sh setupRC.sh
```

# Description
zshの環境はpreztoをフォークしたものを使っています。

> ir77/prezto: The configuration framework for Zsh
> https://github.com/ir77/prezto

vimのstatusline/tablineはlightlineを使って設定しています。
> https://github.com/itchyny/lightline.vim
